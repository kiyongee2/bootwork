1. 회원 가입
   - 이메일 인증코드 확인
   - 회원 정보 입력 : 주소(네이버 주소 변환기)
   - 해외 결제 가능한 카드 입력($1 결재후 환불)
   - 이메일, 비밀번호 로그인 : 콘솔에 로그인
 
2. 리눅스 서버(우분투-ubuntu) 생성
   LightSail 검색
   - create instance
   - Operating System(OS) only -> Ubuntu 22.04 > 5$ 이상

   ** 리눅스 명령어 **
   - clear
   - hostname 변경
   - sudo hostnamectl set-hostname 본인 닉네임
   - exit
   - ubuntu@giyongee2
   - date
   - sudo timedatectl set-timezone Asia/Seoul
   - pwd(현재 위치)

   - khit 디렉터리 생성
     mkdir khit
     ls
     cd khit
     ubuntu@giyongee2:~/khit$
     상위 디렉터리로 이동(한칸띄고) : cd ..

   - Java 설치: openjdk-17
  $ java
  $ sudo apt update
  $ sudo apt-cache search jdk
  $ sudo apt-cache search jdk | grep openjdk-17
  $ sudo apt install openjdk-17-jdk
  $ java --version

   - 인스턴스 : 
      1. 비밀키 다운로드 .pem
      2. 8080포트 open

3. 파일질라로 aws에 업로드 
  프로토콜:SFTP
  호스트: 고정IP
  로그온유형:키파일
  사용자:USER NAME
  키파일: download default key - pem 다운로드후 선택

4.  testweb 프로젝트 생성
    jar 파일로 빌드
   - java -jar testweb.jar 
   - 브라우저 실행(고정IP:8080)

   - 우분투 서버를 종료하면 연결 끊김
   - nohup 명령어 추가
   - nohup java -jar testweb.jar

   ** 프로세스 삭제하기(kill)
   - ps -ef
   - ps -ef | grep testweb
   - kill -9 프로세스번호
